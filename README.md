# AS Prerelease 2025

The AQA AS Computer Science preliminary material for 2025,
and my solutions for several proposed questions.

## How to Use

- This branch (main) contains the original code and prerelease PDF
- The [notes branch](https://github.com/shrub719/as-prerelease/tree/notes)
  contains annotated code and a `utils/notes.md` containing potential questions
- The [question branches](https://github.com/shrub719/as-prerelease/branches/all)
  (under `/q`) contain the code for different possible questions.  
  The `README.md` in the root directory has a description of the question and
  the `.py` file contains all the code that was changed.
